# ORANG Backend API Server

### 프로젝트 실행 
```
git clone https://github.com/2022-EWHA-OSP-5/ORANG_SERVER.git
pip install -r requirements.txt
python app.py
```

### Requirements
```

flask-restx==1.0.3
```
